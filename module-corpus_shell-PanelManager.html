AddMainPanel(panelId, type, position, url, title, zIndex)

purpose: adds a parent/main panel to the list of panels.

  1. Creates a new MainPanelObject using the given parameters. -> module:corpus_shell~PanelManager#GetNewMainPanelObject
  2. Add this panel to the "map" of main panels. -> module:corpus_shell~PanelManager#Panels
  3. Add the panel id to the list of currently used panels (panel ids). -> module:corpus_shell~PanelManager#AddPanelId
  4. Add the panel title to the list of currently used search panel titles. -> module:corpus_shell~PanelManager#AddSearchPanelTitle
Parameters:
Name Type Description

type string The type of pannel for which a main panel should be added.
position Position The postion of the panel.
url url A URL, may be undefined.
title string The intelligable title of the main panel.
zIndex number The height of the main panel object to be added.

OpenNewSearchPanel(config, searchstr)

purpose: Creates a new search panel and registers it with this object.

  1. Fetches the next unique panel id -> module:corpus_shell~PanelManager#GetNewPanelId.
  2. Calculates an initial position as a Position object.
    It uses a cascading effect module:corpus_shell~PanelManager#GetPanelCount and an initial size of 525px x 600px.
  3. Places the new window above all currently visible. -> module:corpus_shell~PanelManager#GetMaxZIndex
  4. Creates a new title for the panel using "Search" as a part of that title. -> module:corpus_shell~PanelManager#GetNewPanelTitle
  5. Converts the internal name to an index if necessary -> module:corpus_shell~PanelManager#GetSearchIdx.
  6. Creates a new module:corpus_shell~Panel object.
  7. Within this new panel create a new search sub panel and display it using module:corpus_shell~Panel#CreatePanel
  8. Add the new panel to this object's "map" of all panels by its unique id. -> module:corpus_shell~PanelManager#PanelObjects
  9. Create a new main panel object for this panel in the map of main panel objects which is one above the created panel -> module:corpus_shell~PanelManager#AddMainPanel
Parameters:
Name Type Argument Description
config number | string The index of the module:corpus_shell~SearchConfig to use or
may be an internal resource name.
searchstr string <optional>
 The search string to execute in the new panel.
